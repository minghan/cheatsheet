IRC Commands
============

If you use weechat:

    $ ./weechat-curses irc://your_nick@irc.freenode.net

If you use irssi:

    $ irssi

Standard Stuff
--------------

/connect irc.freenode.net
/nick your_nick
/msg nickServ identify your_password
/join #cslounge
/part #cslounge
